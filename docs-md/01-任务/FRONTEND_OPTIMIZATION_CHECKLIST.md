## 一、功能优化清单

### P0（演示必做）

- [x] **统一错误提示机制**：所有异常都显示“用户可读提示 + 排查建议 + 原始报错折叠面板”。
- [x] **Tab1 导出能力**：接处警结果支持下载 `Markdown` 与 `JSON`。
- [x] **处理进度反馈**：显示“检索中 → 推理中 → 结果解析中”阶段状态，避免用户误判卡死。
- [x] **关键字段缺失提示**：当地点/人数/武器等为“未知”时给出补录建议。
- [x] **示例按钮状态**：点击示例后高亮当前样例，避免不确定是否填充成功。

### P1（产品体验提升）

- [x] **会话历史面板**：侧边栏保留最近 10 条分析记录（时间、类型、风险、摘要）。
- [x] **一键回填重跑**：历史记录可点击回填输入框并重新分析。
- [x] **复核闭环入口**：增加“人工复核结果”按钮（同意/驳回/需补充）。
- [x] **复核记录导出**：将复核结论与原分析一起导出，形成可提交证据。
- [x] **Tab2 任务化字段**：补充“优先级、时效、责任分工、执行要点”。

### P2（评分加分）

- [ ] **指标看板**：增加平均耗时、成功率、风险分布趋势（按会话统计）。
- [ ] **失败样例收集**：自动记录解析失败/高风险误判样例供后续优化。
- [ ] **RAG 命中可视化**：显示检索到的规程标题与命中条数（可折叠）。
- [ ] **多角色视图**：接警员视图/指挥员视图切换（展示重点不同）。

## 二、美观优化清单

### P0（低成本高收益）

- [x] **统一字体与间距**：标题、正文、卡片间距统一，减少“挤”和“空”不一致。
- [x] **风险色标准化**：紧急红 / 高橙 / 中黄 / 低绿，所有标签和卡片保持一致。
- [x] **卡片视觉统一**：摘要、处置建议、法律依据采用同一卡片样式（圆角、阴影、边框）。
- [x] **按钮视觉层级**：主操作（研判/生成）突出，次操作（导出/调试）弱化。
- [x] **图标体系统一**：同一信息类型固定同一图标，避免混用风格。

### P1（整体质感提升）

- [x] **顶部信息条**：增加系统状态条（模型状态、RAG 状态、当前时间）。
- [x] **分区标题规范**：每个区块使用固定标题格式与分割线。
- [x] **空状态页优化**：无数据时显示引导插图 + 明确下一步操作。
- [x] **加载动画优化**：使用阶段文案替代单一 spinner，提升“系统在工作”的感知。
- [ ] **响应式布局优化**：窄屏自动单列，避免双列压缩导致阅读困难。

### P2（展示级优化）

- [x] **品牌化主题**：统一主色、强调色、图标风格，形成“警擎”视觉识别。
- [x] **数据图表美化**：替换默认图表为更清晰的颜色和标签（含图例与单位）。
- [ ] **动效微交互**：按钮 hover、卡片淡入、结果区平滑展开（不过度动画）。
- [x] **演示模式**：提供“大字号/高对比度”开关，适配投屏答辩场景。
