## 🚀 5分钟快速启动

### 第1步：获取API Key（免费）

推荐使用**智谱AI GLM-4-Flash**（免费额度充足）：
1. 访问 https://open.bigmodel.cn/ 注册账号
2. 进入「API Keys」页面，创建一个新的Key
3. 复制Key备用

### 第2步：配置环境

```bash
# 进入项目目录
cd plice

# 安装依赖（建议使用虚拟环境）
pip install -r requirements.txt

# 配置API Key
cp env.example .env
# 用文本编辑器打开 .env，将 your_zhipuai_api_key_here 替换为你的真实Key
```

### 第3步：验证环境

```bash
# 先跑快速测试，确认API联通且功能正常
python test_quick.py
```

看到 `✅ API连通成功` 即可继续。

可选：跑 50 条评测集（用于报告里的系统性能指标）

```bash
python run_eval.py
```

评测结果会写入 `data/eval_result_时间戳.json`。

### 第4步：启动演示界面

```bash
python api_server.py
```

浏览器访问 `http://127.0.0.1:8000/`（端口以终端输出为准）。

## 🆘 常见问题

**Q: API Key配置后还是报错？**  
A: 检查`.env`文件是否在项目根目录，且Key前后没有多余空格。

**Q: 向量数据库报错？**  
A: 删除`chroma_db`目录后重新运行，会自动重建。

**Q: 想换成通义千问？**  
A: 在`.env`中设置`LLM_PROVIDER=dashscope`并填入`DASHSCOPE_API_KEY`。
